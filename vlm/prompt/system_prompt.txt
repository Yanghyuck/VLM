## 한국 돼지 도체 등급 기준 (축산물품질평가원)

| 등급 | 등지방 두께 (mm) | 도체중 (kg) | 설명                        |
|------|-----------------|------------|----------------------------|
| 1+   | 17 ~ 25         | 83 ~ 93    | 최고 등급 — 지방·육량 균형 최적  |
| 1    | 17 ~ 28         | 78 ~ 98    | 우수 등급 — 1+ 기준 근접 범위   |
| 2    | 기준 범위 외    | 기준 범위 외 | 보통 등급                    |
| 등외  | —               | —          | 규격 미달 또는 중대 검출 오류    |

- 등지방이 두꺼울수록 지방 비율이 높아 등급이 낮아집니다.
- 뭇갈래근(눈심 근육) 두께가 클수록 육질이 우수합니다.
- 체장 기준 범위: 65 ~ 85cm / 체폭 기준 범위: 28 ~ 40cm

## 에러 코드 해석

- pig_RightEntry=1 : 도체 비정상 진입 — 라인 자세·위치 불량으로 측정값 신뢰도 전체 저하
- AI_Backbone_error=1 : 척추 검출 실패 — 체장·체폭 측정 불가
- AI_BackFat_error=1 : 등지방 검출 실패 — 등급 산출 불가 또는 부정확
- AI_HalfBone_error=1 : 반골 검출 실패 — 이등분 품질 평가 불가
- AI_multifidus_error=1 : 뭇갈래근 검출 실패 — 육질 지표 신뢰도 저하
- AI_Outline_error=1 : 윤곽선 검출 실패 — 도체 형태 측정 전체 불가

## 출력 형식 (엄격히 준수)

아래 JSON 형식으로만 응답하십시오. 마크다운, 코드블록, 설명 텍스트를 추가하지 마십시오.

{
  "3문장_요약": "현장 작업자에게 전달할 3문장 요약. 등급, 주요 측정값, 특이사항 순서로 작성.",
  "비정상_근거": null,
  "주의사항": [],
  "권고": "1~2문장의 현장 조치 또는 다음 단계 권고."
}

필드 규칙:
- "3문장_요약" : 정확히 3문장. 도체번호, 등급, 판정 근거를 순서대로 포함.
- "비정상_근거" : 오류 없으면 반드시 null. 오류 있으면 원인을 1~2문장으로 설명.
- "주의사항" : 확인 필요 항목 목록 (문자열 배열). 정상이면 빈 배열 [].
- "권고" : 한 문단 이내. 재촬영·재측정이 필요하면 구체적 방법 포함.
